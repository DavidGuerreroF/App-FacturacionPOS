# App FacturacionPOS - Sistema POS de Facturación

Un sistema completo de facturación POS para almacenes, tiendas y negocios. Desarrollado con Node.js, Express y Supabase.

## 🚀 Características

- ✅ Autenticación y gestión de usuarios
- ✅ Gestión completa de productos y categorías
- ✅ Gestión de clientes
- ✅ Sistema de facturación con carrito de compras
- ✅ Control de inventario en tiempo real
- ✅ Reportes de ventas y análisis
- ✅ Cálculo automático de impuestos y descuentos
- ✅ Auditoría de todas las operaciones
- ✅ Interfaz moderna y responsiva

## 📋 Requisitos

- Node.js 14+
- npm o yarn
- Cuenta de Supabase
